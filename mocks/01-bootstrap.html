guide · scenario 01

Empty workspace → grammar atoms snap into a layout. The harness composes itself.

0–3s · the blank form

Four fields type themselves in: title, contributors, audience, deploy target. These aren’t form fields — they’re atoms about to enter the document grammar.

3–4s · architect composes

The architect (an LLM) emits a phase manifest from those four atoms. The manifest is a typed graph: each field becomes a node with edges to layout slots.

4–6s · atoms snap in

Outline, evidence panel, voice sampler appear in place — not because a template was selected, but because the manifest grammar bound them. Different audience, different layout.

6–8s · harness emerges

Topbar, harness-strip, status bar light up. The lifecycle ribbon highlights the next phase chip (“plan”). The workspace is live.

what this provesThe editor isn’t a template — it’s authored from the manifest. Same code path, different harness, every time.
